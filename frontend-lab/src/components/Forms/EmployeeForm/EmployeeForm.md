  - ✔ Nombres y apellidos
  - ✔ DNI
  - ✔ Cargo
  - ✔ Área
  - ✔ Correo electrónico

- Adicionalmente

  - ✔ Fecha de creacion de usuario (registro automatico)

- Tareas

- A tener en cuenta
